# PyQt Calculator

A simple desktop calculator built with Python and PyQt6.

## Features
- Basic math
- Division-by-zero error handling
- Platform Agnostic

## Requirements
- Python 3.14+
- PyQt6

## Installation

```bash
git clone https://github.com/dood345/pyqt-calculator.git
cd pyqt-calculator
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -e ".[dev]"
```
